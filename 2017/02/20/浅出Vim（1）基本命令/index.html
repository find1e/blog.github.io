浅出Vim（1）基本命令

Feb 20, 2017 | Vim | Hits

Vim 简明手册

当你刚打开VIM编辑器的时候，默认是处于命令模式。在该模式下，你可以通过VIM内建的命令操作文本。
在插入模式下，你可以像在Windows下的记事本已经编辑、输入文本。输入i就可以进入插入模式，按ESC键会退去插入模式，进入命令模式。当你不清楚当前VIM编辑器处于什么模式时，按ESC一定会回到命令模式。下面我们要说的命令都是在“命令模式”下执行的。按下v可以进入可视模式，这样你可以使用h，j，k，l，w，b来操作光标高亮选中文本,按Enter确认选中。

拷贝&复制

命令 说明
yw 拷贝文字
yy 拷贝光标所在行
y$ 拷贝光标所在位置直到行末

删除

命令 说明
x 删除字符
dw 删除双字符
cw 删除双字符然后进入“插入模式”
dd 删除光标所在行
d￥ 删除光标所在位置直到行末

移动光标

命令 说明
k 向上移动光标
j 向下移动光标
h 向左移动光标
l 向右移动光标
w 向右移动一个字
b 向左移动一个字
gg 移动光标到当前行首
G 移动光标到当前文件结尾
$ 移动光标到当前行尾
^ 移动光标到当前行第一个非空字符处

插入文字

命令 说明
i 在光标前
a 在光标后
o 在当前行下新开一行插入
shift + o 在当前行上新开一行插入

退出

命令 说明
:q 没有修改文件时退出
:q! 强制退出
:x 保持文件后退出
:qw 保持文件后退出
:w 保持当前修改，进入”命令模式”

搜索

在命令模式按/+regx,匹配到要查找的内容后，按n向后搜索，按N向前搜索。

shell

输入:sh,可以从VIM暂时进入shell，按exit后返回VIM

撤回

命令 说明
u 撤回上一次操作
ctrl + r 恢复上一步被撤销的操作
